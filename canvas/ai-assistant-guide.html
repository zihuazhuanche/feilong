技能概述

AI Assistant 是基于 Gemini Brain 技能增强开发的智能扩展大脑，提供智能问答、代码生成、文档分析、搜索增强和决策支持等功能。

智能问答

多大脑智能路由，自动选择最佳处理方式，支持上下文对话。

代码生成

支持 Python、JavaScript、Go、Rust 等多种编程语言。

搜索增强

结合网页搜索和本地知识库，提供全面的信息检索。

性能优化

智能缓存、优雅降级、连接池等优化措施。

使用示例

基础问答

# 简单问答
openclaw skill ai-assistant "什么是机器学习？"

# 带上下文的问答
openclaw skill ai-assistant --context "之前的对话" "新的问题"

代码生成

# 生成 Python 代码
openclaw skill ai-assistant --code "python:一个简单的Web服务器"

# 生成 JavaScript 代码
openclaw skill ai-assistant --code "javascript:React组件示例"

高级功能

# 搜索增强
openclaw skill ai-assistant --search "最新的AI研究"

# 查看统计信息
openclaw skill ai-assistant --stats

# 清空对话历史
openclaw skill ai-assistant --clear
