重要更正：PPTX 导出格式说明

在之前的测试中，我错误地描述了 Slidev 的 PPTX 导出功能。经过查阅官方文档，现更正如下：

  • ✅ Slidev 确实支持 PPTX 导出 - 使用 slidev export --format pptx
  • ⚠️ 但是需要注意 - PPTX 文件中的幻灯片是作为图像导出的
  • 📝 文本不可选择 - 无法在 PowerPoint 中编辑文本内容
  • 🗒️ 演讲者备注会包含 - 每张幻灯片的备注会导入到 PPTX 中
  • 🔄 点击动画默认启用 - PPTX 导出默认启用 --with-clicks 选项

建议：如果需要可编辑的文本，请使用 PDF 导出格式。

📁 导出格式对比

📄

PDF 格式

默认导出格式

  • ✅ 文本可选择和搜索
  • ✅ 保持矢量质量
  • ✅ 文件体积较小
  • ✅ 跨平台兼容
  • ⚠️ 不支持动画效果
  • ⚠️ 交互功能受限
# 导出为 PDF slidev export # 导出为 PDF 并包含点击动画 slidev export --with-clicks # 导出特定幻灯片 slidev export --range 1,3,5-10
🖼️

PNG 格式

图像格式导出

  • ✅ 每张幻灯片单独图像
  • ✅ 高质量截图
  • ✅ 适合网页使用
  • ✅ 透明背景支持
  • ⚠️ 文本不可选择
  • ⚠️ 文件体积较大
  • ⚠️ 缩放会失真
# 导出为 PNG 图像 slidev export --format png # 导出特定幻灯片为 PNG slidev export --format png --range 1-5 # 导出透明背景 PNG slidev export --format png --omit-background
💼

PPTX 格式

PowerPoint 兼容格式

  • ✅ PowerPoint 可打开
  • ✅ 包含演讲者备注
  • ✅ 点击动画默认启用
  • ✅ 适合会议演示
  • ⚠️ 幻灯片为图像（文本不可编辑）
  • ⚠️ 文件体积较大
  • ⚠️ 无法修改内容
# 导出为 PPTX (PowerPoint) slidev export --format pptx # 注意：PPTX 中的幻灯片是图像 # 文本不可选择或编辑 # 如需可编辑文本，请使用 PDF 格式 slidev export --format pdf

🚀 实际导出示例

# 完整的工作流程示例： # 1. 创建 Slidev 项目 npm init slidev@latest my-presentation cd my-presentation # 2. 编辑演示文稿 # 编辑 slides.md 文件 # 3. 启动开发服务器预览 npm run dev # 4. 导出为不同格式 # 导出为 PDF (推荐，文本可编辑) slidev export --output my-presentation.pdf # 导出为 PNG 图像 slidev export --format png --output slides # 导出为 PPTX (用于 PowerPoint 播放) slidev export --format pptx --output my-presentation.pptx # 5. 高级选项 slidev export \ --format pdf \ --with-clicks \ --range 1-20 \ --dark \ --timeout 60000 \ --output final-presentation
