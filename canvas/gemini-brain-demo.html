Gemini Brain 技能

将 Google Gemini 作为 OpenClaw 的扩展大脑和搜索引擎

技能概述

这是一个功能强大的 Gemini 集成技能，可以将 Google Gemini 作为 OpenClaw 的智能扩展，提供问答、代码生成、文档分析和决策支持等功能。

智能问答

使用 Gemini 进行深度思考和问题解答，支持上下文对话。

代码生成

支持 Python、JavaScript、Go、Rust 等多种编程语言的代码编写。

实时搜索

结合 Gemini 的搜索能力获取最新信息和网络资源。

文档分析

上传 PDF、Word、Excel 等文件让 Gemini 分析处理。

配置示例

模型 gemini-2.0-flash
温度 0.7
最大输出长度 2048 tokens
支持的语言 Python, JS, Go, Rust, Java, C++
搜索功能 已启用

使用示例

基础使用:

# 通过技能调用
openclaw skill gemini-brain "你的问题"

# 示例: 代码生成
openclaw skill gemini-brain "用Python写一个快速排序算法"

# 示例: 问题分析
openclaw skill gemini-brain "分析一下当前AI行业的发展趋势"

高级功能:

# 带上下文的对话
openclaw skill gemini-brain --context "之前的对话" "新的问题"

# 文件分析
openclaw skill gemini-brain --file document.pdf "总结这个文档"

# 代码审查
openclaw skill gemini-brain --code python "审查这段代码"

快速开始

  1. 获取 Google Gemini API 密钥
  2. 设置环境变量: export GEMINI_API_KEY="your-key"
  3. 安装依赖: pip install google-generativeai
  4. 运行测试: python skills/gemini-brain/quick_test.py
  5. 开始使用 Gemini Brain 技能！
