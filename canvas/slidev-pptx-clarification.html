基于官方文档的正确理解

  • Slidev 确实支持 PPTX 导出
  • PPTX 文件中的幻灯片是图像格式
  • 文本在 PowerPoint 中不可选择 (not selectable)
  • 包含演讲者备注
  • 点击动画默认启用
  • 适合会议演示和分发

📖 官方文档原文

来自 https://sli.dev/guide/exporting#pptx：

# PPTX 导出命令 slidev export --format pptx # 官方说明： "Note that all the slides in the PPTX file will be exported as images, so the text will not be selectable. Presenter notes will be conveyed into the PPTX file on a per-slide basis. In this mode, the --with-clicks option is enabled by default. To disable it, pass --with-clicks false."

关键短语： "text will not be selectable" (文本不可选择)

🎯 PPTX 导出的实际含义

🔍 正确理解 "文本不可选择"

当官方文档说 "text will not be selectable" 时，意思是：

  1. 幻灯片是图像 - 每张幻灯片被渲染为一张图片
  2. 无法用鼠标选择文本 - 在 PowerPoint 中无法高亮或选择文字
  3. 无法直接编辑内容 - 要修改内容需要编辑原始 Markdown 文件并重新导出
  4. 但不影响播放 - 仍然可以在 PowerPoint 中正常播放演示

🚀 使用建议

根据你的需求选择合适的导出格式：

✅ 使用 PPTX 导出的场景：

  • 需要在 PowerPoint 中播放演示
  • 会议或课堂演示
  • 分发不可编辑的演示文稿
  • 包含演讲者备注的演示

✅ 使用 PDF 导出的场景：

  • 需要可搜索和选择的文本
  • 打印或电子文档分发
  • 需要较小文件体积
  • 跨平台兼容性要求高

🔄 工作流程建议：

# 开发阶段：使用 Slidev 网页版 npm run dev # 分发阶段：根据需求导出 slidev export --format pdf # 用于文档分发 slidev export --format pptx # 用于 PowerPoint 演示 slidev export --format png # 用于网页展示 # 重要：保留原始 Markdown 文件以便后续编辑
