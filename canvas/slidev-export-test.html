📤 导出格式测试

📄

PDF 格式

文档格式导出

特点:

  • 文本可搜索和选择
  • 文件体积较小
  • 适合打印和分发
  • 跨平台兼容
slidev export slides.md --output test-presentation.pdf
🖼️

PNG 格式

图像格式导出

特点:

  • 每张幻灯片单独图像
  • 高质量截图
  • 适合网页使用
  • 透明背景支持
slidev export slides.md --format png --output slides
💼

PPTX 格式

PowerPoint 格式导出

特点:

  • PowerPoint 可打开
  • 包含演讲者备注
  • 点击动画默认启用
  • 注意: 幻灯片为图像格式
  • 注意: 文本在 PowerPoint 中不可选择
slidev export slides.md --format pptx --output test-presentation.pptx

#!/bin/bash # 完整的 Slidev 导出测试脚本 # 进入测试目录 cd /home/node/clawd/test-slidev/export-test echo "=== Slidev 导出测试开始 ===" echo "测试时间: $(date)" echo "" # 检查 Slidev 是否安装 if ! command -v slidev &> /dev/null; then echo "安装 Slidev..." npm install -g @slidev/cli fi echo "1. 导出为 PDF..." slidev export slides.md --output test-presentation.pdf if [ -f "test-presentation.pdf" ]; then echo "✅ PDF 导出成功: test-presentation.pdf" echo " 文件大小: $(stat -c%s test-presentation.pdf) 字节" else echo "❌ PDF 导出失败" fi echo "" echo "2. 导出为 PNG..." slidev export slides.md --format png --output slides PNG_COUNT=$(ls slides-*.png 2>/dev/null | wc -l) if [ $PNG_COUNT -gt 0 ]; then echo "✅ PNG 导出成功: 生成 $PNG_COUNT 张图片" else echo "❌ PNG 导出失败" fi echo "" echo "3. 导出为 PPTX..." slidev export slides.md --format pptx --output test-presentation.pptx if [ -f "test-presentation.pptx" ]; then echo "✅ PPTX 导出成功: test-presentation.pptx" echo " 文件大小: $(stat -c%s test-presentation.pptx) 字节" echo " ⚠️ 注意: PPTX 中的幻灯片是图像格式，文本不可选择" else echo "❌ PPTX 导出失败" fi echo "" echo "=== 测试完成 ===" echo "查看生成的文件: ls -la *.pdf *.pptx slides-*.png"
