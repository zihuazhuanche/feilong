使用方式

基础问答

# 简单问答
openclaw skill ai-assistant "你的问题"

# 带上下文
openclaw skill ai-assistant --context "之前内容" "新问题"

# 指定大脑
openclaw skill ai-assistant --brain local_llm "问题"

高级功能

# 代码生成
openclaw skill ai-assistant --code "python:要求"

# 搜索增强
openclaw skill ai-assistant --search "查询内容"

# 文档分析
openclaw skill ai-assistant --file document.pdf "总结"

系统管理

# 查看统计
openclaw skill ai-assistant --stats

# 清空历史
openclaw skill ai-assistant --clear

# 查看帮助
openclaw skill ai-assistant --help
