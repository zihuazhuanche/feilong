🎓 重要发现

1. Slidev 导出依赖

Slidev 导出需要以下关键依赖：

# 必需的核心依赖 npm install @slidev/cli # Slidev CLI npm install @slidev/theme-default # 默认主题 npm install -D playwright-chromium # 浏览器引擎

2. PPTX 导出特性确认

通过官方文档验证：PPTX 导出的是图像格式幻灯片，文本在 PowerPoint 中不可选择。

3. 环境配置要点

  • 需要在项目目录本地安装依赖
  • Playwright 需要浏览器二进制文件
  • Docker 环境可能需要额外配置
